# lapukhou-rags

Claude Code plugin marketplace for local RAG knowledge-base plugins. Each plugin is an Obsidian wiki vault with hybrid retrieval (BM25 + rerank) and a skill that answers questions from it with citations.

## Usage

```
/plugin marketplace add lapukhou/lapukhou-rags
```

## Plugins

| Plugin | Source | Description |
|---|---|---|
| [`rag-jtbd`](https://github.com/lapukhou/rag-jtbd) | GitHub | Jobs-to-be-Done knowledge base — Anthony Ulwick's *Jobs to Be Done: Theory to Practice* (2016): JTBD theory, ODI process, case studies |
| [`rag-octalysis`](https://github.com/lapukhou/rag-octalysis) | GitHub | Octalysis gamification knowledge base — Yu-kai Chou's *Actionable Gamification*: 8 Core Drives, game techniques, White Hat vs Black Hat design, case studies |
| [`rag-growth-web`](https://github.com/lapukhou/rag-growth-web) | GitHub | Web growth knowledge base — web funnels, paid acquisition, onboarding, monetization, funnel analytics, growth process, compliance (WFC course) |
| [`rag-miracle-morning`](https://github.com/lapukhou/rag-miracle-morning) | GitHub | Miracle Morning knowledge base — Hal Elrod's *The Miracle Morning*: SAVERS practices, wake-up strategy, 30-day habit change, Miracle Evening and Miracle Life |

Install with:

```
/plugin install rag-jtbd@lapukhou-rags
/plugin install rag-octalysis@lapukhou-rags
/plugin install rag-growth-web@lapukhou-rags
/plugin install rag-miracle-morning@lapukhou-rags
```
